# Michael Kipchumba — Developer Portfolio

## Folder Structure
```
portfolio/
├── index.html        ← Main page
├── css/style.css     ← All styles (edit accent colours in :root)
├── js/main.js        ← Particles, typewriter, scroll animations
└── README.md
```

## Quick edits
- **LinkedIn URL** — find `https://www.linkedin.com/in/michael-kipchumba-8a74aa288` in index.html and replace
- **Project GitHub links** — replace the `#` hrefs inside `.project-links`
- **More projects** — copy a `.project-card` block and fill in the details
- **Accent colour** — open `css/style.css`, find `--blue:#3b82f6` in `:root` and change

## Deploy to GitHub Pages
1. Go to github.com → New repository → name it `michaelkessio.github.io` → Public
2. Upload `index.html`, `css/` folder, `js/` folder
3. Settings → Pages → Deploy from branch → main / root → Save
4. Visit https://michaelkessio.github.io in ~2 minutes
